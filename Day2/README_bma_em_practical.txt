# BMA-EM practical package

Files included:
- `bma_em_practical_dataset.csv`: synthetic dataset for the practical
- `bma_em_practical_walkthrough.ipynb`: step-by-step notebook for teaching
- `bma_em_practical_walkthrough.py`: plain Python script version

Suggested workflow:
1. Open the notebook.
2. Inspect the individual models.
3. Compare SMA vs BMA-EM.
4. Discuss weights and sigma values.
5. Compute within-model and between-model variance.
6. Finish with the spatial train/test split.
